Question-1: What are the different ways to select an element in the DOM?

We can select HTML element using different ways, see most used.
-First of all we need to write (document.) before getting element

  • 1. document.getElementsByTagName('element name')
  • 2. document.getElementsByClassName('class name')
  • 3. document.querySelectorAll('name')
  • 4. document.querySelector('name')
  • 5. document.getElementById('id name')

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The main difference between these is,

-innerHTML used to get access all of the content of an element including tags.
-innerText used to get the text access of an element.
-textContent used to get all text content of an element even the text content is hide.

Question-3: What is event delegation in the DOM?

By adding a single event on parent element, Event delegation allowes us to handle events triggered on it's child elements,
we can avoid to write seperate event listener for each child,, there is a property called (.target) helps us to write very less code but achieve much more.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process which one propagate child triggered event to upwards through DOM hierarchy, it's triggering untill reach the topmost element.
